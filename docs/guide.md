# 指南

## 快速开始

### 安装

首先，你需要安装我们的UI组件库。你可以通过 `npm`、`yarn`、`pnpm` 来安装它。

```bash
npm install hard-ui
# 或者
yarn add hard-ui
```

### 引入组件

在你的项目中，你可以这样引入并使用一个按钮组件：

```tsx
import React from 'react'
import { Button } from 'hard-ui'

function App() {
	return <Button>点击我</Button>
}

export default App
```
